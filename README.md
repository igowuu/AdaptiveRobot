# Why Adaptive?
- Reduced boilerplate
- Dynamic capabilities
- Quality-of-life improvements
- Request-based arbitration (no subsystem conflicts)
- Request timeouts
- SysID, easy autonomous

Adaptive is a middleground with the benefits of command-based and MagicRobot, aiming to get rid of their limitations.
